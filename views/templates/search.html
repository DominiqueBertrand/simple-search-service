Search

Preview how the service works here, and then dive into the API to put this service to work in your app. Query tips

  • *:* will bring back everything you've indexed
  • [key]:'[value]' will bring back results that match the [key] with the [value]; e.g., town:'ashfield'
  • [key1]:'[value1] OR [key2]:'[value2]' is the syntax for multiple matches
  • Combine free text and faceted search like this: willowbank AND [key]:'[value]'
  • The simple search service assigns default parameter values to your queries, including includes_docs=true and limit=20. Learn more about the API
