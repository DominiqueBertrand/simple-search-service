Get Data

The Simple Search Service indexes CSVs and TSVs

If you don't have data in that format, no problem. We have some data you can use to kick the tires:

  • UK Addresses (CSV)
  • Movies (TSV)

If you have some data dumps of your own, feel free to use those. If you're new to data dumps in our required formats, start with these guides for exporting CSVs and TSVs from:

  • CouchDB / Cloudant
  • MySQL
  • MongoDB
  • PostgreSQL
