Ingredients

The stuff that's baked in.

Runtime
Node.js app running on IBM Bluemix.
Data
IBM Cloudant, running on IBM Bluemix. (If you don't know your Cloudant password, access Cloudant through the app on Bluemix instead.)
Cache (optional)
IBM Compose Redis.
Code
View the app on Github. Of particular note, check out the csv-parse and couchimport npm modules.

To learn more about how these things go together, check out the tutorial and the README.
