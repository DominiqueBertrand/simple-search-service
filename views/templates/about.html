Welcome to the Simple Search Service!

Buckle up, you're about to build an API that can deliver a fully functional, faceted search engine and content discovery service, just by clicking a few buttons.

Excited? So are we.

Let's get into it. We'll do all the heavy lifting for you. Just follow the instructions. But don't worry, this walkthrough is no wizard. If you're a developer and you want to dig in beyond what you see here, we've got you covered:

  • The tutorial spells out how we built it
  • All of the code is on Github

Feel free to fork the code and extend the app. If you have feedback, and want to contribute to master, check out our backlog.

Is this really a service?

Yes.

We use the Simple Search Service to power the search engine in our developer center, and it's also the API that powers content discovery in Bluemix.

If you followed those links, you probably noticed that the content is identical. That's because the content is served from the same Cloudant database, via the same API. The Preview Search and Preview API pages of this app will show you how we used this service to do just that.
